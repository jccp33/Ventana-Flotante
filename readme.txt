VENTANA FLOTANTE

* Este es un ejemplo de una ventana flotante que puede ser posicionada en cualquiera de las cuatro esquinas de tu navegador utilizando la clase CSS w-float junto a alguna de las siguientes clases:

float-top-left
float-top-right
float-bottom-left
float-bottom-right

* El contenido dentro de  <div class="w-float float-bottom-right" id="w-float"></div> le da las características título, contenido y botones de minimizar y maximizar, mismos que pueden ser editado pero puede afectar el diseño de la ventana.

* Esta ventana se mantiene sobre el demás contenido de la página web.
